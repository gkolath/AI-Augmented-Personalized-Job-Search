# AI-Augmented Personalized Job Search

A full-stack Next.js platform that uses AI to personalize your job search — featuring job aggregation, resume optimization, cover letter generation, mock interviews, salary insights, and a career coaching chatbot.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- **Next.js 16** (App Router, Turbopack)
- **Supabase** — auth & database
- **OpenAI** — AI features (resume, cover letter, interview, chat)
- **Tailwind CSS** + shadcn/ui
